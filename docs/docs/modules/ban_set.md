Ban Set Commands
================
## Summary
These commands are prefixed with `ban set`. You can also use `warn set` instead of `ban set`.

## Commands
### *reason*
#### Overloads
**`!ban set reason 10 "placeholder"`**

Sets the reason for the ban.

| Name | Type | Optional |
| --- | --- | --- |
| banID | long | `no` |
| newReason | string | `no` |
