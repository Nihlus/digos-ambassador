Warning Set Commands
====================
## Summary
These commands are prefixed with `warning set`. You can also use `warn set` instead of `warning set`.

## Commands
### *reason*
#### Overloads
**`!warning set reason 10 "placeholder"`**

Sets the reason for the warning.

| Name | Type | Optional |
| --- | --- | --- |
| warningID | long | `no` |
| newReason | string | `no` |
